# 🚀 WEDO - 現代化極簡高效任務管理系統

**WEDO** 是一款基於 Vue 3 與 Vite 構建的高性能、極簡風格任務管理工具。它融合了現代美學（Glassmorphism）與高度自動化的邏輯，旨在為個人提供最流暢的規劃體驗。

## ✨ 核心特色

### 📅 三大維度視圖
- **General (一般清單)**：適合隨手記錄、腦力激盪或不分日期的瑣碎事務。
- **每週任務 (Weekly)**：以週一至週日為導向的循環規劃，具備每週重置基準提示，助您掌握週間節律。
- **每月計畫 (Monthly)**：
  - **動態月曆**：整合全功能月曆，支援跨月份自由切換。
  - **智性修剪**：系統自動保留「當前月份前後各一月」的數據，其餘過期資料自動清理，維持系統極致輕量。

### 📦 暫存區 (Stash)
- 內建常駐的 **Stash 暫存區**，讓您可以隨時暫放尚未決定類型的任務。
- 在任何視圖點擊「清除所有已完成」時，Stash 內的已完成項亦會同步被自動清理。

### 🖱️ 靈動操作體驗
- **全視圖拖拽 (Drag & Drop)**：支援在所有清單與日期格子間自由移動任務。
- **一鍵全域清理**：統一的操作邏輯，一鍵釋放視覺空間。
- **行動優先 (RWD)**：完美的響應式設計，不論在電腦或手機瀏覽器上，品牌 Logo 與 UI 佈局皆能完美呈現。

### 🎨 頂級視覺美學
- **玻璃擬態 UI**：深色背景搭配細緻的毛玻璃效果與漸層邊框，打造沉浸式工作環境。
- **品牌識別**：內建專屬設計的 WEDO 品牌標誌與統一的 Favicon。

## 🛠️ 技術棧
- **框架**：Vue 3 (Composition API)
- **構建工具**：Vite 6
- **狀態管理**：Reactive Stores + LocalStorage 持久化
- **圖標**：Lucide Vue Next
- **樣式**：Vanilla CSS 3 (變數驅動系統)

## 🚀 快速啟動
1. 安裝依賴：`npm install`
2. 啟動開發伺服器：`npm run dev`
3. 構建生產版本：`npm run build`
